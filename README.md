# Proyecto Final de Curso
## Estadística Multivariante orientada al Estudio de Datos Científicos
Construcción de un modelo estadístico para predecir la mortalidad de pacientes con insuficiencia cardíaca congestiva
## Herramientas utilizadas
Se utilizaron datos públicos obtenidos de kaggle.com
Lenguaje utilizado: R
## Metodología
- Análisis exploratorio de las variables
- Análisis de relaciones entre las variables
- Construcción de un modelo Random Forest
- Aplicación del modelo
- Conclusiones
## Archivos del repositorio
- 3.0.R: Código del proyecto
- heart_failure.csv: Datos utilizados
- hf_model.pdf: Memoria y conclusiones
